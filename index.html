Welcome to the NaNoPeg Command Builder!

This site is here to help you use commands on the NaNoPeg Discord server by giving you an easy to use interface so you can tell the bot exactly what you want it to do!

OPTIONAL: Input up to 2 numbers in the range 0-59.
DEFAULT: Start at the next 5 minute increment of the clock (Unless it's less than a 3 minutes away).
This number may not be used exactly, as the bot will try to intelligently adjust the starting point to allow time for people to prepare for the war.
OPTIONAL: Input up the number of minutes to run the War for. Minimum allowed is 1.
DEFAULT: Run the War for 20 minutes.
By default the rating will be set to T or lower.
